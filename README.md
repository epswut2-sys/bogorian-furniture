# Bogorian Furniture

> Rak dinding & furnitur minimalis langsung dari pengrajin lokal

Bogorian Furniture is an Indonesian home decor marketplace connecting local craftsmen with buyers seeking quality wall iron shelves and minimalist furniture. Browse, add to cart, and checkout in minutes — no account needed.

## Stack

| Layer | Service |
|-------|---------|

| Styling | Tailwind CSS + shadcn/ui |

## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in your environment variables
npm run dev
```

### Where to get them

- `DATABASE_URL` — Neon PostgreSQL connection string (Neon dashboard)
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` — Clerk public key for frontend auth (Clerk dashboard)
- `CLERK_SECRET_KEY` — Clerk secret key for server-side auth (Clerk dashboard)
- `NEXT_PUBLIC_CLERK_SIGN_IN_URL` — Clerk sign-in redirect path (Clerk dashboard)
- `NEXT_PUBLIC_CLERK_SIGN_UP_URL` — Clerk sign-up redirect path (Clerk dashboard)
- `STRIPE_SECRET_KEY` — Stripe secret key for payment processing (Stripe dashboard)
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` — Stripe publishable key for frontend checkout (Stripe dashboard)
- `STRIPE_WEBHOOK_SECRET` — Stripe webhook signing secret for order status updates (Stripe dashboard)
- `STRIPE_PLATFORM_FEE_PCT` — Platform commission percentage taken per sale (e.g. 5 for 5%) (Internal config)
- `RESEND_API_KEY` — Resend API key for transactional emails (order confirmation, seller notifications) (Resend dashboard)
- `RESEND_FROM_EMAIL` — Sender email address for all outgoing emails (Resend dashboard)
- `NEXT_PUBLIC_POSTHOG_KEY` — PostHog project API key for analytics (PostHog dashboard)
- `NEXT_PUBLIC_POSTHOG_HOST` — PostHog host URL (PostHog dashboard)
- `SENTRY_DSN` — Sentry DSN for error monitoring (Sentry dashboard)
- `NEXT_PUBLIC_APP_URL` — Public base URL of the deployed application (Vercel environment)
